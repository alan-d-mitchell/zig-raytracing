Setup Instructions:
1. Make sure your Zig ray tracer outputs a PPM file named image.ppm
2. Place this HTML file in the same directory as your image.ppm
3. Run a local HTTP server: python3 -m http.server 8000
4. Open http://localhost:8000 in your browser
5. Run your ray tracer and watch it render in real-time!
